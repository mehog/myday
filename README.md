# NasDan (MyDay)

Digital wedding invitation platform for couples in Bosnia and Herzegovina and the wider Balkans. Multilingual: Bosnian, English, German.

## Local development

```bash
composer setup   # install, migrate, npm build
composer dev     # serve + queue + logs + Vite
```

See [Project context — Build & deployment](docs/architecture/project-context.md#22-build-development--deployment) for environment variables and production notes.
